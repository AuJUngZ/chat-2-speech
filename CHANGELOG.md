# Changelog

## [1.0.0] - 2026-05-08

### Added
- Text-to-Speech (TTS) functionality with cloud and Windows support
- Backend TTS info endpoint to retrieve available voices
- Frontend settings panel for TTS configuration
- Support for Thai, English, and Gemini voices
- Settings panel with tabs for Config and Hotkeys
- Chat popup with Pin and Pass buttons
- Toast notification system with dismiss and retry functionality
